# SSHPass

SSHpass offers you the ability to automatically offer a password via SSH when
you are prompted for it.

the latest version is 1.09

## Install

```bash
git clone https://github.com/RexSi/sshpass
cd sshpass
autoreconf -fi
./configure
make && make install
```
